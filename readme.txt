This Opencart extension allow customers to post topic and replies on forum. A topic can have multiple replies.

Main advantages of this extension

Easy to install, just need to use inbuilt feature of OpenCart - OCMOD (Extension Installer).
Helping hand for Customers - Customers can get reviews on forum related to any product or services.
Helps build brand popularity

Features :
Logged in Customer can post topic on the website.
Admin have to approve the topics to visible on the forum list.
Admin also can create topics.
Customers can reply to any topic.
Email notification to admin to when any customer posts a new topic.
Reply counter
[*] View Counter by topic.
